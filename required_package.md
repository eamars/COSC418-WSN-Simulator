Build OMNeT++
=============

Required packages
-----------------
- bison
- flex
- qt4-default
- tk-dev
- tcl-dev
- default-jre (optional)
- libpcap-dev (optional)
- libopenmpi-dev (optional)


Add PATH to environment variable
--------------------------------
```
if [ -d "$HOME/Workspace/omnetpp-5.0/bin" ]; then
	PATH="$HOME/Workspace/omnetpp-5.0/bin:$PATH"
fi
```
